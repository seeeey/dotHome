그 외의 CSS 네이밍

common.css

common.css는 공통된 Style을 표기한 CSS로 큰 프로젝트 작업시 용이.
( EX. main 관련 css, 폰트 관련 css, 정렬 관련 css 등등 )

layout.css

layout.css는 큰 틀의 Layout을 Style 표기.
( EX. 큰 레이아웃 적용, 공통된 세부적인 style - tab, popup 등등 )

content.css

content.css는 그 밖의 세부적인 Style 표기.
( EX. 그 밖의 디자인 레이아웃을 작업하기 위한 css )

CSS guide

CSS 적용 순서

  • 1. block 요소에서 inline요소로 작성하기

    ( EX. .css { display:block; magin:10px 10px 10xp 10px; padding:10px 10px 10px 10px; width:100px; height:100px; } )

  • 2. font는 inline 요소 뒤 부분에 넣기

    ( EX. .font { display:block; ... height:100px; font-size:20px; color:#eee; font-weight:bold; text-align:center; } )

  • 3. background는 제일 뒤 부분에 넣기 ( 스크롤 생기므로 )

    ( EX. .bg { display:block; ... text-align:center; background:url('/static/images/content/content.jpg) no-repeat; } )

  • 4. 태그의 스타일은 앞 부분에 넣기 ( 고유의 기능을 빨리 캐치할 수 있도록 )

    ( EX. .ul { list-style:none; display:block; ... text-align:center; background:url('/static/images/content/content.jpg) no-repeat; } )

  • 5. 명령은 제일 앞 부분에 넣기

    ( EX. .posi { z-index:3; position:absolute; top:0px; left:0px; display:block; ... } )
